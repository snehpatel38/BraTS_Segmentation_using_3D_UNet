BraTS Segmentation Using 3D UNet

View on GitHub

Procedure

  1. Downloaded the BraTS 3D MRI dataset.
  2. Preprocessed the data (normalization, resizing, etc.).
  3. Built a 3D UNet model using PyTorch.
  4. Trained the model using Dice loss function.
  5. Evaluated the results and visualized predictions.
